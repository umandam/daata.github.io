Do I paste this into README.md?
No. README is not your site. Put this in index.html in the folder GitHub Pages is serving.
How do I change the background + fonts?
Edit the :root variables at the top (colors, sizes), and the Google Fonts link if you want a different font.
Can I add more FAQ items?
Copy one <details>...</details> block and replace the text. That’s it.
How do I use a custom domain?
In your repo’s Pages settings, add your domain. Then add the recommended DNS records at your domain registrar.
